# BalancingCar 嵌入式项目

**平衡车** 是一个基于 STM32F103C8T6 的嵌入式开发项目，主要实现平衡静止，平衡遥控。

## ✨ 功能介绍

- ✅ 功能点 1，微信小程序蓝牙遥控
- ✅ 功能点 2，OLED打印当前位姿
- ...

## 🔧 硬件需求

- **主控芯片**：STM32F103C8T6
- **所需外设**：
  - 0.96寸OLED屏幕
  - 
  - 
- **调试器**：ST-Link

## 💻 软件环境

- **IDE**：Keil MDKv5
- **编译器**：Keil ARM Compiler
