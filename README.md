# BalancingCar 嵌入式项目

**平衡车** 是一个基于 STM32F103C8T6 的嵌入式开发项目，主要实现平衡静止，平衡遥控。

## ✨ 功能介绍

- ✅ 您可用微信小程序蓝牙遥控小车，并通过OLED屏幕观察当前位姿。小车顶部挡板可轻量载荷（如：手机）

## 🔧 硬件需求

- **主控芯片**：STM32F103C8T6
- **所需外设**：
  - 0.96寸OLED屏幕
  - JGB37-520霍尔编码器直流电机 12V
  - MP1584EN DC-DC降压模块 输出 3.3V
  - HC-04D蓝牙模块 支持SPP2.1+BLE4.0
  - MPU6050姿态传感器 6轴imu
  - TB6612电机驱动模块
  - 锂电池组12V输出  
- **调试器**：ST-Linkv2
- **其他材料**：
  - 6P排线 PH2.0转XH2.54 15cm
  - DC-005_5.5-2.5插座
  - 拨动开关SS12D10-3脚
  - 铜柱 M3*60+6【四颗】，M3*30+3【四颗】，M3*11+3【一颗】
  - 亚克力板*2 15*8*2mm
  - 螺钉螺母20+ 类型M3

## 💻 软件环境

- **IDE**：Keil MDKv5
- **编译器**：Keil ARM Compiler

## ⌨️ 二次开发

- **方向**：PCB预留了四个排针接口，可连接超声波模块，以达到避障效果；PCB共有4个按键，本项目仅使用了一个，您可根据自己的需要为各个按键设置特定功能。

## 🚀 快速开始（克隆 + 使用）

1. **克隆仓库**
   ```bash
   git clone https://github.com/isAlexYoung/BalancingCar-.git
2.  **接线与烧录**：
    - 接线：STM32通过杜邦线与STlink链接，再插入PC（如：母对母杜邦线接vcc接vcc）
    - 编译与烧录：在keil中先编译再烧录
3.  **启动与遥控**:
    - 启动：初始状态小车无反应，按下按键1，小车启动保持平衡
    - 遥控：微信打开小程序：蓝牙串口【江协科技】，连接蓝牙即可用摇杆遥控小车（可调整参数）
